2026-05-29

I'm starting a project. I'll write about it here.

Why here? Socials? Nah. I'm good. Substack? Maybe. But not yet.

As my friend Mike would say, "start with the simplest thing that can work."

I'm going off grid. Hand-written. Static html. No tricks, no weapons. Skill against skill alone.

Pure browser defaults (aside from max-width—I'm not an animal). If I add style sheets over time, we'll see that progression in successive entries.

No likes. No subscribes. No comment section.
You want me? I'm right here.
Might even start a newsletter.

It feels silly to hard code a static site in 2026. Platforms are better if you want to be heard and make money. Those aren't my goals right now.

Writing forces me to think clearly. I want a small community of people who make things.

I worry some people may find it inconvenient or self-indulgent. I do have to start somewhere, though.

For me, it starts here.
